## Ausgewählte Projekte

### QuantzAI

**QuantzAI** ist eine von Yones Alizai privat entwickelte, Machine-Learning-basierte Forschungs- und Softwareplattform zur Analyse von Forex- und Rohstoffmärkten.

Das Projekt verbindet die Verarbeitung historischer Marktdaten, Machine-Learning-Modelle, zeitbasierte Validierungsverfahren, Risikomanagement, Paper-Trading und eine grafische Desktop-Oberfläche.

Ein besonderer Schwerpunkt liegt auf der realistischen Bewertung von Strategien. Dabei werden nicht nur Modellmetriken, sondern auch Handelskosten und die tatsächliche Performance innerhalb zeitlich getrennter Testphasen berücksichtigt.

Zu den technischen Bestandteilen gehören:

- Verarbeitung und Validierung historischer Marktdaten
- Feature Engineering für Zeitreihen
- Machine-Learning-Modelle mit LightGBM
- Walk-Forward-Validierung
- Trend- und Mean-Reversion-Analysen
- Risiko- und Positionsmanagement
- Paper-Trading
- Anbindung an MetaTrader 5
- Audit-Logging mit SQLite
- grafische Desktop-Oberfläche mit PySide6
- automatisierte Tests mit pytest

**Technologien:** Python, LightGBM, scikit-learn, pandas, PyArrow, PySide6, MetaTrader 5, SQLite, Loguru und pytest

Der vollständige Quellcode, interne Modelle, Strategielogik, Konfigurationen und vertrauliche Projektdaten bleiben privat. Öffentlich präsentiert werden ausschließlich ausgewählte Benutzeroberflächen, Videos, Projektbeschreibungen und technische Einblicke.

### Protocolbridge

**Protocolbridge** ist eine von Yones Alizai entwickelte IIoT-Middleware zur Verbindung unterschiedlicher industrieller Kommunikationsprotokolle.

Das Projekt führt **Modbus**, **OPC UA** und **MQTT** über eine gemeinsame REST-Schnittstelle zusammen. Geräte- und Systemdaten können dadurch zentral verarbeitet und über ein Echtzeit-Dashboard dargestellt werden.

Ziel des Projekts ist es, die Kommunikation, Integration und Überwachung unterschiedlicher industrieller Systeme zu vereinfachen.

Zu den Schwerpunkten gehören:

- Verbindung unterschiedlicher IIoT-Protokolle
- zentrale REST API
- Verarbeitung von Gerätedaten
- Echtzeit-Monitoring
- Visualisierung technischer Zustände
- Vereinheitlichung verschiedener Schnittstellen
- Integration industrieller Systeme

**Technologien und Themen:** IIoT, Modbus, OPC UA, MQTT, REST API, Systemintegration und Echtzeit-Dashboard

Der vollständige Quellcode bleibt privat. Eine ausgewählte öffentliche Demonstration zeigt die grundlegende Funktionsweise des Projekts.

### IT Integrator App

Die **IT Integrator App** ist eine Desktop-Anwendung zur Unterstützung typischer Aufgaben aus der IT-Systemintegration.

Die Anwendung wird mit **Electron** und **React** entwickelt. Ziel ist es, verschiedene technische Werkzeuge und Arbeitsabläufe in einer zentralen, übersichtlichen Desktop-Oberfläche zusammenzuführen.

Zu den vorgesehenen Schwerpunkten gehören:

- zentrale Oberfläche für technische Aufgaben
- Unterstützung wiederkehrender IT-Arbeitsabläufe
- strukturierte Darstellung von Systeminformationen
- plattformübergreifende Desktop-Nutzung
- Erweiterbarkeit durch zusätzliche Werkzeuge und Module

**Technologien:** Electron, React, JavaScript, HTML und CSS
