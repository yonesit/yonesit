## Projekte

### QuantzAI

QuantzAI ist ein privates Forschungs- und Softwareprojekt zur datenbasierten Analyse von Forex- und Rohstoffmärkten.

Das Projekt verbindet historische Marktdaten, Machine-Learning-Modelle, zeitbasierte Validierung, Risikomanagement, Paper-Trading und eine grafische Desktop-Oberfläche.

**Technologien:** Python, LightGBM, scikit-learn, pandas, PySide6, MetaTrader 5, SQLite und pytest

Der vollständige Quellcode, die internen Modelle und die Strategielogik bleiben privat. Öffentlich gezeigt werden ausgewählte Videos, Oberflächen und Projektinformationen.

[GitHub-Showcase](https://github.com/yonesit/quantzai-showcase) · [Projektseite](https://yonesit.github.io/quantzai.html) · [Video-Demo](https://youtu.be/-rbE4jQJ4iU)

> QuantzAI ist ein privates Forschungsprojekt und keine Finanzberatung.

### Protocolbridge

Protocolbridge ist eine IIoT-Middleware zur Verbindung unterschiedlicher industrieller Kommunikationsprotokolle.

Das Projekt verbindet Modbus, OPC UA und MQTT über eine gemeinsame REST API und stellt Daten in einem Echtzeit-Dashboard dar.

**Schwerpunkte:** IIoT, Systemintegration, REST API und Echtzeit-Monitoring

[Öffentliche Demo ansehen](https://github.com/yonesit/protocolbridge-demo)

### IT Integrator App

Die IT Integrator App ist eine Desktop-Anwendung für Aufgaben aus der IT-Systemintegration.

Sie wird mit Electron und React entwickelt und soll verschiedene technische Werkzeuge und Arbeitsabläufe in einer zentralen Oberfläche zusammenführen.

**Technologien:** Electron, React, JavaScript, HTML und CSS
